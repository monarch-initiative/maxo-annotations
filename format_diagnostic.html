MAxO Diagnostic Annotations: Format

The maxo_diagnostic_annotations.tsv file contains six columns separated by tabs (TSV format).

  • hpo_id

  • hpo_label

  • predicate_id: is_observable_through or is_prenatally_observable_through

  • maxo_id

  • maxo_label

  • creator_id

Each row in the table specifies a diagnostic modality that is able to ascertain the HPO term in question. The following table shows a few examples
